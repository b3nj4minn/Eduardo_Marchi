## Despliegue
Suba el contenido de esta carpeta tal como está a cualquier hosting estático.

## Formulario de contacto
El formulario funciona sin backend: al enviar, abre el cliente de correo del usuario con el asunto y el mensaje prearmados.
Si más adelante quiere integrarlo con Formspree, Resend, Netlify Forms o backend propio, basta con reemplazar el JS de `assets/js/site.js`.
